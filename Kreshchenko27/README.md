# Memory Training App - Практична робота №27

Цей застосунок є проєктом для тренування пам'яті, модернізованим у межах практичної роботи №27.

## Виконані покращення:
- **Qt Resource System**: Впроваджено файл `resources.qrc`, що дозволяє вбудовувати ресурси (стилі) безпосередньо у виконуваний файл програми.
- **Централізовані стилі (QSS)**: Дизайн застосунку винесено у файл `style.qss`. Замість жорсткого програмування стилів у C++, застосовано глобальний підхід, що забезпечує візуальну узгодженість усіх екранів.
- **Покращення UX**: Додано візуальні стани кнопок (`:hover`, `:disabled`), що покращує зворотний зв'язок для користувача.
- **Адаптивність**: Проєкт оптимізовано з використанням `layout` менеджерів для коректного відображення на різних розмірах вікон.
